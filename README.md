# 定制项目

## 框架
* ui: semantic-ui-react
* tools: fabric.js, ruler.js
* frame: react (webpack3)

## 功能
1. 添加文字
2. 添加图片、图片上传
3. 自定义按钮（旋转、缩放、删除、文字编辑）
4. 更换背景色、背景图
5. 标尺线、标识线
6. 画布对象控制（置底、置顶、中心对齐、向上一层、向下一层）
7. 清空
8. 保存图片至本地
9. 本地缓存
10. ...

## 安装
* git clone https://Mango3403@bitbucket.org/Mango3403/custom.git
* cd custom
* npm install

## 开始
* npm run start

## 构建
* npm run build
